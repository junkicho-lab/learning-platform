## 🎯 에세이 스토리 작성 3단계

### Step 1: 주제 선정 및 재료 수집

**방법:** 축약본에서 핵심 메시지를 뽑고, 관련 예제노트를 모읍니다.

```
예시: "기록이 상담을 바꾼다"
├── 축약본 5장: 패턴 분석 방법론
├── 김민준.md: 화장실빌런 사례
├── 2026-03-07.md ~ 2026-05-09.md: 시간순 기록
└── 화장실빌런의비밀.md: 완성된 에세이 참고
```

### Step 2: 시간순 배열 (스토리 뼈대)

**김민준 스토리 예시:**

| 시점 | 사건 | 감정/해석 | 출처 |
|:--|:--|:--|:--|
| 3/7 | 영어 시간 화장실 15분 | "짜증 + 걱정" | 2026-03-07.md |
| 3/14 | 또 화장실, 패턴 의심 | "게으름인가?" | 2026-03-14.md |
| 3/21 | 3주 연속, 패턴 확정 | "회피구나!" | 2026-03-21.md |
| 3/28 | 면담 "영어가 무서워요" | 원인 파악 | 김민준.md |
| 4/2 | 스크립트 제공 시작 | 해결 시도 | 김민준.md |
| 5/9 | "안 무서웠어요" | 첫 성공 | 2026-05-09.md |
| 11월 | 75점 최고기록 | 성장 완료 | 김민준.md |

### Step 3: 살 붙이기 (에세이 완성)

**구조:**
1. **도입:** 문제 상황 (화장실빌런 등장)
2. **전개:** 패턴 발견 과정 (기록의 힘)
3. **위기:** 원인 파악의 어려움
4. **해결:** 맞춤 지원 시작
5. **결말:** 성장과 깨달음

## 📝 에세이 스토리 초안 템플릿

### 제목: [주제를 담은 제목]

```markdown
# [제목]

## 1. 시작 - 문제의 발견

[첫 장면 묘사 - 데일리노트에서 가져오기]

> "2교시 영어, [[김민준]] 또 화장실 감. 15분 소요."
> — 2026년 3월 7일 데일리노트

[교사의 첫 반응과 감정]

---

## 2. 전개 - 패턴의 발견

[반복되는 상황 나열]

| 날짜 | 상황 |
|:--|:--|
| 3/7 | ... |
| 3/14 | ... |

[패턴을 인식하는 순간]

> "이건 게으름이 아니었다. 회피였다."

---

## 3. 클라이맥스 - 진짜 이유

[면담 장면 - 상담 기록에서 가져오기]

> "영어 시간이 무서워요."

[교사의 깨달음]

---

## 4. 해결 - 작은 시도

[구체적인 지원 방법]

- 듣기 스크립트 미리 제공
- 쉬운 문제부터 성공 경험

---

## 5. 결말 - 성장

[변화의 증거]

> "쌤, 저 듣기 70점 나왔어요!"

[교사의 성찰과 메시지]

---

## 💡 이 이야기가 말하는 것

[핵심 메시지 1~2문장]
```

## 🎨 추천 에세이 주제 4선

### 1. 화장실빌런의 비밀 ⭐ (완성 예시 있음)

**재료:**
- 축약본: 프롤로그, 5장
- 예제노트: `김민준.md`, `화장실빌런.md`, `화장실빌런의비밀.md`
- 데일리노트: 3/7, 3/14, 3/21, 3/28, 4/2, 5/9

**핵심 메시지:** "감으로 하는 상담 vs 데이터로 하는 상담"

---

### 2. 조용한 아이를 발견하기까지

**재료:**
- 축약본: 10장 (그래프 뷰)
- 예제노트: `정수아.md`, `리더십.md`, `갈등조정.md`
- 데일리노트: 9/12, 9/18, 9/25, 10/15, 11/5

**핵심 메시지:** "문제아는 자동으로 기록되지만, 모범생은 잊혀진다"

**스토리라인:**
1. 1학기 기록 0건 - 조용해서 놓침
2. 9월 그래프 뷰에서 '고립된 점' 발견
3. 의식적 관찰 시작
4. 조용하지만 영향력 있는 리더십 발견
5. 생기부에 구체적 문장 작성 가능

---

### 3. 공격적인 아이의 진짜 이유

**재료:**
- 축약본: 5장, 7장
- 예제노트: `강도현.md`, `가정환경.md`, `방어기제.md`, `분노조절.md`
- 데일리노트: 3/14, 3/20, 4/2, 5/15

**핵심 메시지:** "공격적인 아이가 아니라, 방어적으로 먼저 공격하는 습관이 생긴 아이"

**스토리라인:**
1. 3월 다툼 발생
2. 가정환경 파악 (부모 갈등)
3. 행동 패턴 이해
4. 맥락을 아는 상담
5. 스승의 날 편지 "저도 고치고 싶어요"

---

### 4. 기록이 나를 지켜준 날

**재료:**
- 축약본: 11장
- 예제노트: `학부모 민원 대응법.md`, `김민준.md` (6월 민원), `정수아.md` (11월 민원)

**핵심 메시지:** "기록은 방패다"

**스토리라인:**
1. 학부모 민원 전화
2. 당황하는 순간
3. 기록 검색 → 증거 제시
4. 원만한 해결
5. 기록의 가치 재발견

## 🔧 실전 작성 프로세스

### 1단계: 재료 모으기
```bash
# 관련 파일 목록 만들기
1. 축약본에서 해당 챕터 읽기
2. 예제노트에서 관련 인물/키워드 노트 열기
3. 데일리노트에서 시간순 기록 추출
```

### 2단계: 타임라인 작성
```markdown
| 날짜 | 사건 | 감정 | 행동 | 결과 |
|:--|:--|:--|:--|:--|
| ... | ... | ... | ... | ... |
```

### 3단계: 초안 작성
- 타임라인을 따라 이야기 형식으로 풀어쓰기
- 각 장면 사이에 교사의 고민, 시도, 깨달음 넣기

### 4단계: 퇴고
- 불필요한 부분 삭제
- 핵심 메시지 강조
- 독자 관점에서 읽어보기

## ✨ 핵심 요약

1. **축약본** = 방법론 (어떻게 기록하고 분석하는가)
2. **예제노트** = 실제 사례 (구체적인 기록과 결과)
3. **에세이** = 둘을 엮어 하나의 이야기로 만든 것

> "글쓰기는 무에서 유를 창조하는 게 아니에요. 
> 흩어진 재료를 엮어서 하나의 이야기로 만드는 것입니다."
