📘 옵시디언 설치 가이드

10분 만에 시작하는 교사용 기록 시스템

Part 1. 옵시디언 설치 10분

"씨앗을 심는 것은 10분, 그러나 그 나무는 평생 그늘을 드리운다"

옵시디언 설치는 정원에 첫 나무를 심는 것입니다. 구글 드라이브라는 비옥한 토양에 보관함이라는 씨앗을 심으면, 자동으로 물(백업)이 공급됩니다.

1.1 시작하기 전에

✅ 준비물 체크

  • 인터넷 연결
  • 구글 드라이브 폴더 ("옵시디언" 폴더)
  • 10분 시간

1.2 다운로드 (3분)

Step 1

공식 사이트 접속

브라우저 주소창에 입력 (크롬, 엣지 등):

obsidian.md
Step 2

"Download for Windows" 버튼 클릭

💡 참고:

  • Windows: ObsidianSetup-1.10.x.exe
  • Mac: Obsidian-1.10.x.dmg
  • (숫자는 버전에 따라 다름)
Step 3

파일 다운로드 대기

  • 다운로드 시간: 약 1-2분
  • 파일 크기: 약 100MB

브라우저 하단에 다운로드 진행 표시가 나타납니다. 100% 완료될 때까지 기다리세요.

1.3 설치 (5분)

Step 1

다운로드한 파일 실행

ObsidianSetup-1.10.x.exe 더블클릭
Step 2

설치 진행

  • 자동으로 설치됨
  • 약 30초 소요
  • 설치 완료!

1.4 설치 후 첫 화면

옵시디언을 처음 실행하면 아래 언어를 English → 한국어로 변경합니다.

그러면 영어 메뉴가 한글 메뉴로 바뀝니다.

1.5 새 보관함 생성 (5분)

💡 보관함이란?

보관함 (Vault) = 옵시디언 파일들이 저장되는 폴더

비유:

  • 보관함 = 금고
  • 노트 = 금고 안의 문서들
중요: 보관함은 구글 드라이브에 만들면 좋습니다.
(이유: 컴퓨터 구분없이 어디서나 접근)
Step 1

보관함 이름 입력

이름 입력: "최초의노트" (원하는 이름 가능)

💡 이름 추천

✅ 좋은 이름:

  • 최초의노트
  • 내노트
  • 선생님의기록
  • MyVault

❌ 피해야 할 이름:

  • 띄어쓰기 있는 이름
  • 특수문자 있는 이름
Step 2

보관함 위치 선택

이게 가장 중요합니다!

  1. "위치" 옆에 "탐색" 버튼 클릭
  2. 폴더 선택 창(탐색기)에서 "문서" 폴더 선택
  3. "옵시디언" 폴더 선택
  4. 확인
최종 경로: C:\Users\User\Documents\옵시디언\최초의노트\
Step 3

생성 버튼 클릭

  • "Create" 버튼 클릭
  • 잠시 대기 (1-2초)
  • 옵시디언 메인 화면이 열림!

1.6 첫 노트 작성 (2분)

🎉 설치 완료! 이제 첫 노트를 작성해봅시다.

Step 1

새 노트 만들기

Ctrl + N (Windows)
⌘ + N (Mac)

또는 왼쪽 상단 📄 아이콘 클릭

Step 2

노트명 / 노트 제목 입력

노트명 입력: 맨 위에 "무제"가 선택된 채 보입니다. 타이핑하면 바로 노트명이 변경됩니다.

노트 제목 입력:

# 왜 옵시디언 인가?
Step 3

내용 작성

가운데 편집 화면에 입력:

# 251201_월_문법수업_성찰

과거시제 문법 수업
설명 위주로 진행했더니 학생 참여가 적었다
다음에는 게임 요소를 넣어봐야겠다
Step 4

자동 저장 확인

💡 중요: 옵시디언은 자동 저장됩니다!

  • ✅ Ctrl+S 필요 없음
  • ✅ 타이핑하는 즉시 저장
  • ✅ 저장 버튼 누를 필요 없음

1.7 화면 구성 이해하기

📁 왼쪽: 파일 목록

  • 내가 만든 노트들
  • 폴더 구조
  • 클릭하면 노트 열림

📝 가운데: 편집 화면

  • 노트 작성/수정
  • 여기서 대부분의 작업

📊 오른쪽: 사이드바

  • 그래프뷰 (나중에 배움)
  • 백링크 (나중에 배움)
  • 추가 정보

Part 2. 마크다운 기초 15분

"단순함이 궁극의 정교함이다" - 레오나르도 다빈치

마크다운은 젓가락입니다. 처음엔 포크와 나이프가 편해 보이지만, 익숙해지면 젓가락 하나로 모든 요리를 먹을 수 있습니다. #, ##은 측정자, #은 견출지, [[]]는 실과 바늘. 3가지 도구로 문서라는 옷을 짜는 것입니다.

2.1 마크다운이 뭔가요?

일반적인 방법 (한글/워드)

  1. 텍스트 드래그
  2. 굵게 버튼 클릭
  3. 크기 선택
  4. 색상 변경

→ 번거로움!

마크다운 방법

  1. **굵게**만 입력
  2. 자동으로 굵게 표시!

→ 초간단!

💡 핵심 개념

마크다운 = 기호로 글 꾸미기

비유: 이모티콘 ^^ → 😊 처럼, **굵게** → 굵게

✅ 마우스 안 써도 됨

✅ 빠름 (버튼 찾을 필요 없음)

✅ 어떤 프로그램에서든 동일

✅ 10년 후에도 열림

2.2 딱 5가지만 알면 끝

교사에게 필요한 건 딱 5가지입니다. 나머지는 지금 당장 배울 필요 없어요!

A. 제목 만들기

입력

# 큰 제목 (제일 큼)
## 중간 제목
### 작은 제목

결과

큰 제목

중간 제목

작은 제목

💡 # 뒤에 띄어쓰기 필수! #의 개수에 따라 크기가 달라집니다.

B. 링크 걸기 (가장 중요!)

입력

[[링크할 노트 이름]]

결과

김민준 ← 클릭하면 김민준 노트로 이동

💡 대괄호 2개 사이에 이름. 파일이 없어도 됨 (자동 생성). 이게 옵시디언의 핵심!

C. 태그 달기

입력

#학생지도 #2학년 #김민준

결과

#학생지도 #2학년 #김민준

💡 태그는 노트를 분류하는 라벨입니다. 태그를 누르면 그 태그가 달린 노트만 모아볼 수 있습니다.

D. 목록 만들기

입력

- 첫 번째 항목
- 두 번째 항목
- 세 번째 항목

결과

  • 첫 번째 항목
  • 두 번째 항목
  • 세 번째 항목

💡 -(하이픈)이나 *(별표)를 문장 앞에 붙이면 목록이 됩니다. 번호 목록은 1. 사용

E. 체크박스

입력

- [ ] 할 일 (안 함)
- [x] 완료된 일 (완료)

결과

  • 할 일 (안 함)
  • 완료된 일 (완료)

💡 목록 기호 뒤에 [ ]를 추가하면 체크박스가 됩니다. 클릭하면 체크/해제 전환!

F. 굵게, 기울임

입력

**굵게**
*기울임*
***굵게 + 기울임***

결과

굵게

기울임

굵게 + 기울임

Part 3. 링크의 힘

"모든 것은 연결되어 있다. 우주의 거미줄을 건드리면 전체가 진동한다" - 인디언 속담

[[학생이름]]은 포털입니다. 회색 링크는 에러가 아니라 미래로 가는 문. 아직 존재하지 않는 방으로 통하는 문을 미리 만들어두는 것입니다.

3.1 옵시디언 링크가 뭔가요?

💡 핵심 개념

링크 = 노트와 노트를 연결하는 다리

비유: 웹사이트 링크처럼 클릭하면 다른 노트로 이동

✅ 노트 사이 이동이 즉시

✅ 관련 정보가 자동으로 모임

✅ 검색 안 해도 됨

✅ 정리가 자동으로 됨

3.2 링크 걸기 실습

Step 1

링크 문법

[[노트명]]

예시:

오늘 [[김민준]] 학생과 상담했다.
[[수학 수업]] 준비가 필요하다.
[[생활기록부]] 작성 시작!
Step 2

실제로 링크 걸어보기

  1. 새 노트 만들기: Ctrl + N
  2. 노트명: 2025-10-12
  3. 내용 입력:
# 오늘의 상담 기록

오늘 [[김민준]] 학생과 상담을 진행했다.

3.3 링크 색깔의 의미

회색 링크

[[새 파일]]

의미: 이 이름의 노트가 아직 없음

동작: 클릭하면 파일 자동 생성!

파란색 링크

[[기존 파일]]

의미: 이 이름의 노트가 이미 있음

동작: 클릭하면 그 파일로 이동

💡 중요: 회색도 정상!

많은 초보자가 "회색이면 잘못된 거 아냐?" 라고 착각합니다.

  • ❌ 아닙니다!
  • ✅ 회색은 정상입니다!
  • ✅ 클릭하면 파일 자동 생성
  • ✅ 이게 옵시디언의 마법!

3.4 링크 실습: 학생 노트 만들기

Step 1

링크 클릭하여 노트 생성

Ctrl + 클릭으로 링크를 클릭하면 새 노트가 생성됩니다.

Step 2

학생 프로필 작성

생성된 노트에 학생 정보를 입력합니다:

# 김민준

## 기본 정보
- 반: 2-3
- 번호: 15번
- 특징: 과학 실험 좋아함

## 관찰 기록
- 2025-10-12: 광합성 실험 적극 참여
Step 3

뒤로 가기 - 링크 색깔 확인

Alt + ← 또는 왼쪽 상단 화살표로 돌아가면 링크가 파란색으로 변합니다!

Step 4

양방향 연결 확인

김민준 노트에서 백링크를 확인하면 수업일지가 자동으로 연결되어 있습니다!

3.5 링크 효과: Before vs After

❌ Before (링크 없이)

  1. "김민준 어디 적었더라?"
  2. 파일 10개 뒤지기
  3. 검색창에 "김민준" 입력
  4. 3개 파일에 흩어진 내용
  5. 수동으로 복붙

→ 10분 소요

✅ After (링크 사용)

  1. 김민준 노트 클릭
  2. 백링크에 모든 내용 자동 표시
  3. 끝!

→ 10초 소요

절약: 9분 50초 × 학생 30명 = 5시간

3.6 링크 고급 기능 (선택)

📌 별칭 사용하기

문법

[[파일명|표시 이름]]

예시

[[김민준|민준이]]와 상담

화면에는: 민준이와 상담

📌 제목 링크

문법

[[파일명#제목]]

예시

[[김민준#관찰 기록]]

→ 김민준 노트의 "관찰 기록" 부분으로 바로 이동

🎁 링크 치트시트

기능 문법/단축키
기본 링크 [[노트 이름]]
별칭 링크 [[파일|별칭]]
제목 링크 [[파일#제목]]
링크 따라가기 Ctrl + 클릭
뒤로 가기 Alt + ←

💡 많이 걸수록 좋음!

🎯 자주 묻는 질문

Q1: 링크를 너무 많이 걸면 안 좋나요?

❌ 아닙니다! ✅ 많이 걸수록 좋음! 10개, 100개, 1000개 상관없음

Q2: 띄어쓰기 있는 이름도 되나요?

✅ 됩니다! [[김 민준]] ✅ [[중2-3반 과학실험]] ✅

Q3: 나중에 이름 바꾸면?

✅ 옵시디언이 자동으로 모든 링크 업데이트! 걱정하지 마세요!

Q4: 실수로 링크 잘못 걸었는데?

✅ 그냥 지우면 됨. 파일은 안 지워짐 (따로 삭제 필요)
