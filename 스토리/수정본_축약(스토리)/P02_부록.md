# 부록: 옵시디언 빠른 시작 가이드

설치부터 세팅까지 10분이면 완성됩니다.

1단계는 다운로드입니다. 2분이면 됩니다. obsidian.md에 접속해서 Download를 클릭하고 설치 파일을 더블클릭해서 완료합니다.

2단계는 볼트 만들기입니다. 1분이면 됩니다. Create new vault를 클릭하고, Vault name에 '나의교무수첩'을 입력하고, Create를 클릭합니다.

3단계는 한글 설정입니다. 1분이면 됩니다. 설정에서 언어를 한국어로 바꿀 수 있습니다.

---

교사 필수 단축키 TOP 5를 알려드리겠습니다.

1위는 Ctrl + O입니다. 빠른 열기 기능으로, 학생 이름으로 노트를 찾을 때 씁니다.

2위는 Ctrl + Shift + F입니다. 전체 검색 기능으로, 학부모 전화 시 기록을 찾을 때 씁니다.

3위는 Ctrl + G입니다. 그래프 뷰 기능으로, 놓친 학생을 발견할 때 씁니다.

4위는 Ctrl + L입니다. 체크박스 기능으로, 할 일 목록을 만들 때 씁니다.

5위는 Ctrl + Z입니다. 되돌리기 기능으로, 실수했을 때 씁니다.

---

기능 번역 사전입니다.

Vault는 볼트로, 노트들이 저장되는 폴더입니다. Daily Note는 데일리 노트로, 매일 자동 생성되는 오늘의 기록장입니다. Link는 링크로, 노트끼리 연결하는 대괄호입니다. Backlink는 백링크로, 나를 언급한 다른 노트들입니다. Tag는 태그로, 분류용 스티커입니다. Embed는 임베드로, 다른 노트 내용을 끼워넣는 것입니다. Graph View는 그래프 뷰로, 노트 연결을 시각화하는 것입니다.

---

링크로 만들 만한 키워드를 알려드리겠습니다.

학생 관련으로는 학생이름, 화장실빌런, 지각, 발표 같은 것들이 있습니다.

수업 관련으로는 모둠활동, 피타고라스 정리, 월요병, 비오는날 같은 것들이 있습니다.

행사와 업무 관련으로는 체육대회, 수학여행, 학부모상담주간, 가정통신문 같은 것들이 있습니다.

교육 개념으로는 자기효능감, 성장마인드셋, 학습된 무기력 같은 것들이 있습니다.
