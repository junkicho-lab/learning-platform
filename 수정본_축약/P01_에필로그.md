## 💫 점 → 선 → 면 → 입체 → 우주

**3월 첫날, 선생님은 점 하나를 찍었습니다.**

```
오늘 준호가 발표할 때 목소리가 떨렸다.
```

**4월, 점들이 선으로 연결되기 시작했습니다.**

```
[[이준호]] 검색
→ 3월: 목소리 떨림
→ 5월: 쉬운 질문에 대답 성공!
→ 6월: 스스로 손들고 발표
```

**9월, 선들이 면이 되어 패턴을 보여주었습니다.**

- [[비오는날]] → 우리 반은 차분해짐
- [[월요병]] → 1교시는 활동 수업 필수

**12월, 면들이 입체가 되어 선생님을 지켰습니다.**

- 학부모 민원: 1년 치 지도 기록이 방패
- 생기부 마감: 1년 치 #행특 자동 수집

**그리고 이 기록들은 선생님만의 교육 철학이 되었습니다.**
