## 이 장을 마치면...

| 예전 방식 | 지금 방식 |
|:--|:--|

> **"기록은 찾아가는 게 아니라, 찾아오게 만드는 거예요."**

## 새로 나오는 용어

| 용어 | 뜻 |
|:--|:--|
| **백링크 (Backlinks)** | '나를 언급한 다른 노트들'의 목록. 김민준 노트를 열면, 김민준을 언급한 모든 데일리 노트가 자동으로 모여 있어요. |
| **연결된 언급** | 백링크의 한국어 표현. 오른쪽 사이드바에서 확인할 수 있습니다. |
| **사이드바** | 화면 왼쪽 또는 오른쪽에 있는 보조 패널. 파일 목록, 백링크 등을 보여줍니다. |

## 교실 현장의 에피소드

민준이 어머니에게서 전화가 왔습니다. "민준이가 영어를 너무 싫어해요. 학교에서 무슨 일 있었나요?"

기억을 더듬어봅니다. 3월에 뭔가 적었던 것 같은데... 결국 "확인해보고 연락드릴게요"라며 전화를 끊고, 10분간 폴더를 뒤졌지만 찾지 못합니다.

---

## 4-1. 클릭 한 번으로 순간 이동하기

2장에서 우리는 중요한 단어에 대괄호 `[[ ]]`를 씌우는 법을 배웠습니다.

```
3교시에 [[김민준]]이 또 화장실 간다고 나갔다.
```

이제 그 **파란색 글씨**를 클릭해볼 차례입니다.

### 💡 옵시디언에서 해보기

데일리 노트에 이렇게 적혀 있다고 상상해보세요.

> 오늘 영어 시간에 [[김민준]]이 "저 어차피 못해요"라며 학습지를 덮었다.

이 파란색 `[[김민준]]`을 클릭하면, 민준이의 개인 노트로 바로 이동합니다. 마치 도라에몽의 '어디로든 문'처럼요.

처음 클릭하면 **텅 빈 화면**이 나타납니다. 당황하지 마세요!

새로운 방 문을 열고 들어간 것뿐이에요. 지금 보이는 빈 화면은 `[[김민준]]`이라는 이름의 새 방(노트)입니다. 아까 계시던 방(데일리 노트)은 그대로 있어요.

### 💡 화면 분할 꿀팁: `Ctrl + Click`

새 방으로 이동하는 게 불안하다면, **`Ctrl` 키를 누른 채로** 파란 글씨를 클릭해보세요.

화면이 **좌우로 분할**되면서, 왼쪽에는 원래 쓰던 노트가, 오른쪽에는 새로 열린 노트가 나란히 보입니다.

| 클릭 방법 | 결과 |
|:--|:--|
| 그냥 클릭 | 현재 화면이 새 노트로 바뀜 (순간이동) |
| `Ctrl + 클릭` | 화면이 분할되어 둘 다 보임 (창문 열기) |

## 4-2. 백링크: 내가 찾지 않아도 모여 있는 기록

자, 이제 백링크를 알아볼게요.

`[[김민준]]` 노트를 열었습니다. 처음엔 빈 페이지였죠.

하지만 **오른쪽 사이드바**를 보세요. **'백링크(Backlinks)'** 또는 **'연결된 언급'**이라는 패널이 있습니다.

거기에 뭐가 보이시나요?

```
연결된 언급 (3개)

- 2026-03-12 : "오늘 영어 시간에 [[김민준]]이 '저 어차피 못해요'라며..."
- 2026-04-05 : "[[김민준]]에게 쉬운 1번 문제만 풀게 했더니..."  
- 2026-05-20 : "[[김민준]] 어머니 전화. 학원 끊겠다고..."
```

내가 찾지도 않았는데, 3월, 4월, 5월에 민준이를 언급한 모든 기록이 **알아서 모여 있습니다.**

### 🔊 "메아리" 비유로 이해하기

백링크는 **산에서 외친 메아리**와 비슷해요.

선생님이 3월 12일에 "김민준~!"이라고 외쳤습니다(기록했습니다). 그 소리는 산 어딘가에 흡수되어 있다가, 오늘 `[[김민준]]` 노트를 열자마자 **"저 여기 있어요~!"** 하고 메아리쳐 돌아온 거예요.

| 전통적 방식 | 옵시디언 방식 |
|:--|:--|
| 내가 폴더를 뒤져서 기록을 **찾아감** | 기록이 **자동으로 모여** 있음 |
| "3월 폴더 어디 있지?" | `[[김민준]]` 클릭 한 번 |
| 10분 검색 | 바로 확인 |

---

## 4-3. 점이 선이 되는 순간

### 💫 흩어진 기록이 연결되면

지금까지 선생님은 매일 **점**을 찍어왔습니다.

```
3월 12일: [[김민준]] "저 어차피 못해요" ← 점
4월 05일: [[김민준]] 쉬운 문제 성공    ← 점  
5월 20일: [[김민준]] 학원 끊겠다고     ← 점
```

이 점들은 각각 다른 날짜의 데일리 노트에 흩어져 있었습니다.

하지만 `[[김민준]]`을 클릭하는 순간, 이 점들이 **하나의 타임라인으로 연결**됩니다.

```
김민준의 타임라인 (백링크가 모아줌)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
3월 ●━━━━━━ 4월 ●━━━━━━ 5월 ●━━━━━━ 6월
    "못해요"    성공!      학원문제    지금
```

**패턴이 보이기 시작합니다.**

- 3월: 자신감 없음 → "어차피 못해요"
- 4월: 쉬운 문제 성공 → 표정 밝아짐
- 5월: 학원 스트레스 → 회피 시작
- 6월: 어머니 전화 → "학원 안 가겠대요"

**여기서 알 수 있는 것:**

> "아! 민준이는 '어려운 것 = 실패'라는 공식이 머릿속에 있구나. 쉬운 문제로 성공 경험을 쌓아주니까 밝아졌는데, 학원에서 또 어려운 걸 시키니까 회피하는 거야!"

## ✏️ 실습: 지금 바로 해보기

### 🎯 미션: 김민준의 1년을 한자리에 모으기

**Step 1.** 검색창(`Ctrl + O`)을 열고 학생 이름을 입력합니다.

**Step 2.** 오른쪽 사이드바의 **'연결된 언급'**을 확인합니다.

**Step 3.** 기록들을 훑어보며 **패턴**을 발견합니다.

**Step 4.** 학생 노트 본문에 핵심만 정리합니다.

### 예시: 학생 노트 본문

```markdown
# 김민준

## 📌 핵심 패턴
- 어려운 과제 → 회피 ("어차피 못해요")
- 쉬운 과제 성공 → 자신감 상승

## 💡 효과적인 전략
- 난이도를 낮춰 '작은 성공' 반복시키기
- 실패해도 괜찮다는 정서적 지지

## 🔗 관련 기록
(아래 백링크 패널에서 자동으로 모임)
```

### 🖇️ 꼬리에 꼬리를 무는 연결 제안

> **Q. 백링크에 기록이 너무 많아서 정신없어요. 어떻게 정리하나요?**
> 
> **A.** 백링크는 **'원재료 창고'**라고 생각하세요. 다 읽을 필요 없어요.
> 
> - **본문** = 정제된 요약 (핵심만 정리)
> - **백링크** = 날것의 증거 (필요할 때 참고)
> 
> 중요한 기록만 골라서 본문에 옮기면 됩니다.

> **Q. 민준이 노트에 다른 키워드도 연결할 수 있나요?**
> 
> **A.** 물론이에요! `[[자기효능감]]` 같은 키워드도 링크해두세요.
> 
> 나중에 `[[자기효능감]]` 노트를 열면, 민준이뿐 아니라 **비슷한 패턴의 학생들**이 한꺼번에 보입니다.

## 🛟 박 선생님의 Q&A 코너

> 이 코너는 컴퓨터가 어려운 선생님들을 위한 공간입니다.

### Q1. "파란 글씨를 눌렀더니 화면이 백지가 됐어요! 아까 쓴 거 다 날아간 거야?"

**A.** 선생님, 놀라지 마세요. 아까 쓰던 글이 삭제된 게 아닙니다!

새로운 방 문을 열고 들어간 것뿐이에요. 아까 계시던 방(데일리 노트)은 그대로 있습니다.

**돌아가는 방법 3가지:**
1. 화면 왼쪽 위의 `←` 화살표 클릭
2. 상단 탭에서 아까 쓰던 노트 클릭
3. `Ctrl + O` → 노트 제목 검색

### Q2. "오른쪽에 백링크 패널이 안 보여요!"

**A.** 당황하지 마세요. 숨어있을 뿐이에요.

1. 화면 **오른쪽 끝**에 작은 화살표(`>`)가 있으면 클릭해서 펼치세요.
2. 상단 메뉴 → **보기(View)** → **백링크 패널 열기**
3. 단축키: `Ctrl + Shift + L` (Mac은 `Cmd + Shift + L`)

### Q3. "왔다 갔다 하기가 무서워요. 화면이 바뀌면 헷갈려요."

**A.** `Ctrl`을 누른 채로 클릭하세요!

화면이 좌우로 분할되어서 **두 노트를 나란히** 볼 수 있어요. 마치 책상에 서류 두 장 펼쳐놓은 것처럼요.

### Q4. "이거 꼭 해야 해요? 그냥 검색하면 안 돼요?"

**A.** 검색도 됩니다! 하지만 차이가 있어요.

| 방식 | 소요 시간 | 결과 |
|:--|:--|:--|
| 폴더 뒤지기 | 10분 | 찾을 수도, 못 찾을 수도 |
| 검색 | 1분 | 찾을 수 있음 |
| **백링크** | **바로** | **이미 모여 있음** |

백링크는 "찾는 것"이 아니라 "이미 모여 있는 것을 보는 것"이에요.

# 📎 한페이지 핵심요약

## 핵심 기능: 백링크 (Backlinks)

**한 줄 정의:** 과거에 `[[김민준]]`이라고 적어둔 모든 기록이, 민준이 노트를 열면 자동으로 모여 있는 기능.

## 3가지 핵심 포인트

1. **클릭 = 순간이동**
   - `[[학생이름]]` 클릭 → 해당 학생 노트로 이동
   - `Ctrl + 클릭` → 화면 분할 (두 노트 동시 보기)

2. **백링크 = 자동 모음**
   - 내가 찾지 않아도 기록이 모여 있음
   - 10분 검색할 필요 없이 바로 확인

3. **점 → 선 전환**
   - 흩어진 기록들이 연결되어 패턴이 보임
   - 학생의 행동 맥락을 파악할 수 있음

## 예전 방식 → 지금 방식

| 예전 방식 | 지금 방식 |
|:--|:--|
| "확인해보고 연락드릴게요" | "기록을 보니..." 바로 답변 |
| 파일 10분 뒤지기 | 클릭 한 번으로 확인 |
| 기록만 있고 연결 없음 | 패턴 발견 가능 |

## 실습 요약

1. `Ctrl + O` → 학생 이름 검색
2. 오른쪽 사이드바 → 백링크 패널 확인
3. 패턴 발견 → 본문에 핵심만 정리
