# 분노조절

## 📌 정의

화가 났을 때 감정을 적절히 조절하지 못하고 **폭발적으로 표출**하는 패턴. 욕설, 물건 던지기, 신체 접촉 등으로 나타남.

**핵심 구분:**
| 구분 | 설명 |
|:--|:--|
| 분노 자체 | 자연스러운 감정 (문제 아님) |
| 분노 표출 방식 | 조절이 안 되는 것 (지도 대상) |

> "화내지 마"가 아니라 "화가 났을 때 다르게 표현하는 법"을 가르쳐야 함

---

## 🔗 관련 학생

*백링크에서 자동 수집됨*

| 학생 | 상황 | 배경 |
|:--|:--|:--|
| [[강도현]] | 4/2 체육 시간 욕설, 교우 충돌 | [[방어기제]] + [[가정환경]] |

### [[강도현]] 분노조절 사례

**발생 패턴:**
```
3/14: 류승민과 어깨빵 → 고성
4/02: 체육 축구 경기 중 욕설
5/15: 스승의 날 편지 "저도 제 성격 고치고 싶어요"
```

**학생 자기 인식:**
> "저도 고치고 싶어요. 근데 그 순간엔 멈출 수가 없어요."

**배경 분석:**
- [[가정환경]]: 아버지의 고성에 노출 → 큰 소리 = 위협
- [[방어기제]]: 위협 감지 → 선제 공격으로 자기 보호
- 분노조절 문제처럼 보이지만, **실제로는 방어 반응**

## 💡 대응 전략

### 폭발 직전 (예방)

**신호 포착:**
| 신호 | 대응 |
|:--|:--|
| 얼굴 붉어짐 | "잠깐 쉬다 올래?" |
| 주먹 쥠 | 물리적 거리 확보 |
| 목소리 높아짐 | 다른 학생들 분리 |

**즉각 개입:**
```
"도현아, 지금 많이 화났구나. 잠깐 복도에서 물 마시고 올래?"
```
→ 폭발 전에 상황에서 분리

### 폭발 중 (위기 관리)

| 원칙 | 이유 |
|:--|:--|
| 안전 확보 최우선 | 다른 학생 대피 |
| 말 최소화 | 자극 줄이기 |
| 물리적 거리 유지 | 신체 접촉 피하기 |
| 기다리기 | 흥분이 가라앉을 때까지 |

**하지 말 것:**
- ❌ 큰 소리로 제압하기
- ❌ "진정해!" 반복
- ❌ 신체 접촉으로 제지 (위험)

### 폭발 후 (후속 처리)

**1단계: 진정 후 대화 (30분~1시간 뒤)**
```
"아까 많이 힘들었지? 지금은 좀 괜찮아?"
```

**2단계: 감정 인정**
```
"화가 나는 건 당연해. 
근데 그렇게 표현하면 도현이도 힘들잖아."
```

**3단계: 대안 행동 탐색**
```
"다음에 그런 기분이 들면 어떻게 하면 좋을까?"
- 심호흡 3번
- 그 자리 떠나기
- 주먹 쥐고 10까지 세기
- 선생님한테 신호 보내기
```

**4단계: 기록**
```markdown
[[강도현]] 4/2 분노 폭발 후속 상담
- 트리거: 체육 시간 신체 접촉
- 대안 행동 합의: 화나면 손 들고 나가기
- 다음 관찰: 체육 시간 주의
```

## 📊 장기 지도 전략

### 트리거 파악

| 질문 | 기록할 것 |
|:--|:--|
| 언제 폭발하는가? | 교시, 과목, 상황 |
| 누구와 있을 때? | 특정 학생, 특정 관계 |
| 어떤 상황에서? | 신체 접촉, 큰 소리, 지적당했을 때 |

### 성공 경험 기록

```
9/15: 체육 시간 부딪혔는데 참고 나가서 진정함 → 칭찬!
11/20: 후배 도와주는 모습 → #행특
```

→ 변화 과정을 기록해두면 **생기부에 성장 스토리**로 활용 가능

## 📊 생기부 문장 예시 (성장 후)

> "초반 감정 조절에 어려움을 보였으나, 꾸준한 자기 성찰을 통해 긍정적으로 변화함. 갈등 상황에서 스스로 자리를 피해 감정을 조절하는 모습을 보이며, 후배를 배려하는 태도에서 성장이 돋보임."
