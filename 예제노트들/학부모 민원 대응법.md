# 학부모 민원 대응법

## 📌 핵심 원칙

> **"기억은 왜곡되지만, 기록은 그대로 남아 있다."** - 11장

1. **기록이 있으면 근거가 된다** - 변명이 아니라 팩트로 말할 수 있음
2. **감정 싸움을 피한다** - 감정 대 감정은 양쪽 다 상처
3. **즉각 방어하지 않는다** - "아닙니다!"는 변명으로 들림
4. **통화 후 반드시 기록한다** - 다음 민원에 대비

---

## 📌 3단계 팩트 전환 대화법

### 1단계: 감정 수용 (3초)

> **"어머니/아버지, 많이 걱정되셨겠습니다."**

- 먼저 학부모의 감정을 **인정**
- 이것만으로 대화의 온도가 내려감
- "그러셨군요", "속상하셨겠어요"

### 2단계: 확인 요청 (5초)

> **"정확한 상황을 말씀드리기 위해 기록을 확인해 봐도 될까요?"**

- 방어하지 말고 **확인하겠다**고 함
- 대화의 주도권이 교사에게 넘어옴
- 이 사이에 `Ctrl+Shift+F` → 학생 이름 검색

### 3단계: 팩트 제시 (검색 후)

> **"기록을 보니 3월 15일, 22일, 29일에 이렇게 지도했습니다."**

- **날짜**와 **구체적인 행동**을 말함
- "분명히 했습니다" ❌ → "몇 월 며칠에 이렇게 했습니다" ✅
- 필요시 문자로 정리해서 발송

## 📋 유형별 대응

### 유형 1: "우리 애만 혼내요"

**학부모 주장:**
> "선생님이 우리 애만 미워하시는 것 같아요. 맨날 혼난대요."

**실제 상황:**
- 해당 학생이 규칙 위반을 자주 함
- 다른 학생도 동일하게 지도함
- 아이가 집에서 과장해서 말함

**대응 방법:**
1. 감정 수용: "어머니, 아이가 속상해했나 봐요."
2. 확인 요청: "기록을 확인해 볼게요."
3. 팩트 제시: "3월에 OO이 지각으로 주의받은 건 2회입니다. 같은 날 다른 학생 3명도 동일하게 지도했습니다."
4. 추가: "혹시 OO이가 특별히 힘들어하는 부분이 있을까요?"

### 유형 2: "수업 시간에 방치해요"

> [[이준우]] 아버지 사례 (11장)

**학부모 주장:**
> "준우가 수업 시간에 자는데 왜 안 깨우십니까? 포기하신 거예요?"

**대응 방법:**
1. 감정 수용: "아버님, 준우 성적이 걱정되셨겠습니다."
2. 확인 요청: "기록을 확인해 보겠습니다."
3. 팩트 제시:
```
검색: [[이준우]]

결과:
- 3월 15일: 준우 깨워서 뒤에 서 있게 함
- 3월 22일: 개별 면담 진행
- 3월 29일: 부모님께 알림장 발송
- 4월 5일: 보건실 상담 연계
```
> "한 달간 4번의 지도 기록이 있습니다. 가정에서 수면 시간 확인이 가능하실까요?"

**결과:** 학부모 태도 변화, 가정 협조 약속

### 유형 3: "자기만 미워해요" (화장실)

> [[김민준]] 어머니 사례 (11장)

**학부모 주장:**
> "민준이가 울면서 왔어요. 선생님이 자기만 미워한대요. 화장실도 못 가게 했다면서요?"

**대응 방법:**
1. 감정 수용: "어머니, 민준이가 많이 속상했나 봐요."
2. 확인 요청: "기록을 확인해 볼게요."
3. 팩트 제시:
```
검색: [[김민준]]

결과:
- 3/7: 2교시 영어 화장실 (15분 소요)
- 3/14: 또 화장실, 보건실 요청
- 3/21: 수행평가 중 화장실 요청 → 끝나고 가도록 안내
- 3/28: 상담 - 영어 듣기평가 불안 확인
- 4/5: 듣기 스크립트 미리 제공, 안정됨
```
> "3월 21일 수행평가 중에 시험 끝나고 가도록 안내드린 적이 있습니다. 평소에는 다 보내드렸고요. 사실 민준이가 매주 목요일 영어 시간에만 화장실을 가서 상담했더니, 듣기평가가 힘들었더라고요. 지금은 스크립트를 미리 주고 있고, 잘 적응하고 있습니다."

**결과:** "아... 그런 거였군요. 민준이가 왜곡해서 말한 것 같네요."

### 유형 4: "성적이 떨어졌어요"

**학부모 주장:**
> "OO이 성적이 왜 이렇게 떨어졌어요? 수업을 제대로 안 하시는 거 아니에요?"

**대응 방법:**
1. 감정 수용: "성적이 떨어져서 걱정되셨겠습니다."
2. 확인 요청: "OO이 수업 상황을 확인해 볼게요."
3. 팩트 제시: 출석, 과제 제출, 수업 태도 기록
4. 협력 제안: "가정에서 학습 시간 확인해주시고, 학교에서도 보충 지도하겠습니다."

## 💡 민원 예방 팁

### 1. 선제적 소통
- 문제 발생 시 **학부모보다 먼저 연락**
- "오늘 이런 일이 있었습니다" 알림장/문자
- 아이가 집에서 왜곡해서 말하기 전에 팩트 전달

### 2. 지도 시 증거 남기기
- 지도 후 **데일리 노트에 한 줄**
- `[[학생이름]]` 링크로 연결
- 나중에 검색 가능하게

### 3. 칭찬도 기록
- 칭찬 기록이 쌓이면 균형 잡힌 설명 가능
- "혼내기만 한다"는 민원에 칭찬 기록 제시
