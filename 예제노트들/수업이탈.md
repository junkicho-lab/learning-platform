# 수업이탈

## 📌 정의

수업 중 자리를 비우는 모든 행동의 상위 키워드. 화장실, 보건실, 교무실 심부름 등 다양한 명목으로 수업에서 이탈하는 패턴.

**하위 유형:**
| 유형 | 설명 |
|:--|:--|
| [[화장실빌런]] | 화장실을 핑계로 이탈 |
| [[보건실투어]] | 아프다며 보건실로 이탈 |
| 심부름형 | "교무실 가도 돼요?" |
| 물 마시기형 | "물 마시고 올게요" |

## 💡 대응 전략

### 1단계: 유형 파악
```
이탈 행동 발생
    │
    ├── 화장실 → [[화장실빌런]] 노트에 기록
    ├── 보건실 → [[보건실투어]] 노트에 기록
    └── 기타 → 이 노트에 기록
```

### 2단계: 빈도 체크
| 빈도 | 판단 | 대응 |
|:--|:--|:--|
| 월 1~2회 | 정상 범위 | 관찰만 |
| 주 1회 | 패턴 의심 | 기록 시작 |
| 주 2회 이상 | 회피 행동 가능성 | 상담 필요 |

### 3단계: 공통점 찾기

기록이 3~4개 모이면 아래 질문으로 분석:
- 같은 **요일**인가?
- 같은 **교시**인가?
- 같은 **과목**인가?
- 같은 **활동 유형**인가? (발표, 시험, 모둠 등)

### 4단계: 본질적 질문

| ❌ 피할 질문 | ✅ 효과적인 질문 |
|:--|:--|
| "왜 자꾸 나가니?" | "이 수업이 힘든 거 있어?" |
| "꾀병 아니야?" | "혹시 불안한 거 있어?" |
| "다음부터 참아" | "어떻게 하면 덜 힘들까?" |
