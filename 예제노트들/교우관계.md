# 교우관계

## 📌 정의

학생들 간의 친구 관계 전반을 다루는 키워드. 친한 관계, 갈등 관계, 고립 상태 등 모든 대인관계 양상을 포함.

**기록할 상황:**
- 친구와의 다툼/화해
- 특정 학생과 자주 어울림
- 혼자 있는 모습 관찰
- "친구들이랑 사이가 안 좋아요" 같은 호소


## 💡 대응 전략

### 갈등 상황 발생 시

**1단계: 즉각 분리**
- 흥분 상태에서는 대화 불가
- "일단 떨어져 있어. 5분 뒤에 이야기하자"

**2단계: 개별 청취**
- 각자 따로 이야기 듣기
- 상대방 앞에서 진술받지 않기

**3단계: 맥락 파악**
- 이전에도 있었던 갈등인가?
- [[강도현]] 백링크 확인 → 패턴 있는지 체크

**4단계: 원인 탐색**
| 표면적 문제 | 숨은 원인 가능성 |
|:--|:--|
| "걔가 먼저 시비 걸었어요" | 평소 쌓인 감정? |
| "일부러 그랬잖아요" | 오해? 예민한 상태? |

### 고립 학생 발견 시

- [[관심필요학생]] 목록에 추가
- 그래프 뷰에서 연결 상태 확인
- 1:1 대화 시도 → [[라포형성]]

## 📊 그래프 뷰 활용 (10장)

| 관찰 포인트 | 의미 |
|:--|:--|
| 두 학생이 자주 함께 언급됨 | 관계가 있음 (긍정 또는 갈등) |
| 가운데 있는 학생 | 교우관계 활발 |
| 가장자리에 떨어진 학생 | 고립 가능성 → 관심 필요 |

**예시:** [[강도현]] - [[류승민]] 사이에 선이 있다면 → 둘이 뭔가 있었다는 뜻
