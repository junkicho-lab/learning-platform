# 과제제출

## 📌 정의

수업 과제, 수행평가, 독후감, 탐구보고서 등을 **기한 내에 제출**하는 것. 생기부에서 **성실성과 책임감**을 보여주는 핵심 지표.

> 과제 제출율 100%는 당연한 게 아니다. 기록해두면 "성실하게 모든 과제를 제출함"이라고 쓸 수 있다.

---

## 📊 추적 방법

### 데일리 노트 기록 예시

```markdown
- 과제 확인: [[정수아]] 독후감 제출 완료. [[과제제출]]
- 수행평가: [[박서준]] 탐구보고서 기한 내 제출. [[과제제출]]
- 미제출: [[이준우]] 영어 워크북 미제출. [[과제미제출]]
```

### 검색 방법 (12월 생기부 작성 시)

```
검색: [[정수아]] [[과제제출]]
결과: 12건 (전체 과제 12개 중 12개 제출) → "100% 제출"
```

### 기록 팁
- **제출**과 **미제출** 모두 기록 (균형 있는 파악)
- 특별히 잘한 과제는 내용도 메모
- `#세특` 태그 함께 붙이기

## 🔗 관련 학생

### [[정수아]] - 제출율 100%
- 독후감 4회 전부 제출
- 수행평가 전 과목 기한 내 제출
- 탐구보고서 내용도 충실

### [[박서준]] - 제출율 95%
- 대부분 성실하게 제출
- 1회 지각 제출 (사유 있음)

### [[김민준]] - 제출율 향상 (70% → 90%)
- 1학기: 영어 과제 자주 미제출
- 2학기: [[자기효능감]] 향상 후 제출율 상승

### [[이준우]] - 관찰 필요
- 제출율 60%대
- 특히 영어, 수학 과제 미제출 빈번
- [[학부모상담]] 시 가정 협조 요청

## 📝 생기부 문구 예시

### 세부능력특기사항 (교과)
- "모든 과제를 **기한 내에 성실하게 제출**하며 책임감 있는 학습 태도를 보임."
- "수행평가에서 충실한 내용으로 과제를 완성하여 제출함."
- "탐구보고서 작성 시 자료 조사를 꼼꼼히 하여 완성도 높은 결과물을 제출함."

### 행동특성 및 종합의견
- "과제 제출에 있어 한 번도 빠지지 않는 **성실함**을 보임."
- "맡은 바 책임을 다하는 태도로 모든 과제를 기한 내 완수함."

### 성장 스토리 (김민준 예시)
- "초반 과제 제출에 어려움을 보였으나, 2학기에는 **90% 이상의 제출율**을 보이며 책임감이 향상됨."

## ⚠️ 미제출 학생 관리

### 미제출 패턴 추적

```
검색: [[이준우]] [[과제미제출]]
결과: 5건 → 패턴 확인 (영어, 수학에 집중)
```

### 대응 방법
1. 3회 이상 미제출 시 **개별 면담**
2. 패턴 파악 (특정 과목? 특정 시기?)
3. [[학부모상담]] 시 가정 협조 요청
4. 필요시 `#관찰필요` 태그
