# 기자재고장

## 📌 현상

TV 안 켜짐, 인터넷 끊김, 프린터 고장, 컴퓨터 먹통 등 **수업 중 기자재가 작동하지 않는** 상황. 준비한 수업이 틀어지는 순간.

> "PPT 띄우려는데 TV가 안 켜져요. 20명이 나를 쳐다보고 있고..."

## 💡 대응 전략

| 상황 | 추천 수업 방식 |
|:--|:--|
| TV 안 됨 | 칠판 수업, 모둠 토론으로 전환 |
| 인터넷 끊김 | 오프라인 활동 (손들기 퀴즈, 역할극) |
| 프로젝터 고장 | 유인물 배포, 교과서 중심 수업 |
| 컴퓨터 먹통 | USB 백업 활용, 옆 반 기기 빌리기 |

### 예방 및 대비

1. **USB 백업 필수** - 수업 자료는 클라우드 + USB 이중 저장
2. **플랜 B 준비** - 기자재 없이도 가능한 활동 1개 준비
3. **수업 전 점검** - 쉬는 시간에 TV 켜보기
4. **행정실 연락처** - 급할 때 바로 연락

### 플랜 B 예시

| 원래 수업 | 플랜 B |
|:--|:--|
| PPT 강의 | 칠판 + 교과서 |
| 영상 시청 | 교사가 줄거리 설명 + 토론 |
| 카훗 퀴즈 | 손들기 OX 퀴즈 |
| 인터넷 검색 활동 | 모둠 토론 |
