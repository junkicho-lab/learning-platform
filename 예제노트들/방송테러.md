# 방송테러

## 📌 현상

수업 중 **갑자기 방송이 나와서** 흐름이 끊기는 상황. 안내 방송, 긴급 공지, 방송부 실수 등.

> "딱 집중할 때 '띵동~' 하고 방송이 나오면 애들 집중력이 순식간에 날아가요."

## 💡 대응 전략

| 상황 | 추천 수업 방식 |
|:--|:--|
| 방송 후 | 5초 멈춤 → "자, 다시!" 신호 |
| 중요한 수업 | 가능하면 방송 적은 시간대 배치 |
| 발표 중 방송 | "괜찮아, 이어서 해볼까?" 격려 |
| 반복되는 방송 | 유머로 넘기기 "또 왔네~" |

### 방송 후 집중력 회복 루틴

1. 방송 끝나면 **5초 침묵** (정리 시간)
2. "자, 어디까지 했지?" 복기
3. 짧게 요약하고 이어서 진행
4. 방송이 너무 길면 → 잠깐 휴식 후 재시작

### 예방책

- 중요한 발표/평가는 **방송 적은 시간대**에 배치
- 교무실에 "이 시간엔 방송 자제 요청" 가능한지 확인
