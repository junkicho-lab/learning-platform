# 수업실패

## 📌 정의

계획대로 진행되지 않거나, 학생 반응이 저조했거나, 목표를 달성하지 못한 수업. **실패한 수업일수록 기록할 가치가 있다.** 왜냐하면 "왜?"라는 질문을 남기고, 그 답을 찾는 과정이 전문성 성장이기 때문.

> "성공한 수업은 잊혀지지만, 실패한 수업은 기록하면 교훈이 된다." - 8장

---

## 📊 추적 방법

### 데일리 노트 기록 예시

```markdown
## 3교시 - 수학 (피타고라스 정리)

[[수업실패]] 😞

**상황:** 모둠활동 시켰는데 무임승차 심각
**원인 추정:** 역할 분담 없이 "알아서 하세요" 했음
**다음에 시도할 것:** [[직소모형]] 적용

#수업반성
```

### 기록 필수 항목
1. **무슨 일이 있었나** (상황)
2. **왜 그랬을까** (원인 추정)
3. **다음에 어떻게 할까** (개선안)

### 검색 방법

```
검색: [[수업실패]] [[모둠활동]]
결과: 실패 → 원인 → 해결책 흐름 확인
```

## ⚠️ 실패 기록 시 주의사항

### ❌ 피할 것
- "그냥 망했다"로 끝내기
- 원인 분석 없이 넘어가기
- "다시는 안 해야지"로 포기

### ✅ 해야 할 것
- **"왜?"** 질문하기
- **"다음엔?"** 대안 적기
- 재도전 후 **결과 기록**
