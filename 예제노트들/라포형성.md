# 라포형성

## 📌 정의

학생과 교사 사이의 **신뢰 관계**를 구축하는 과정. 상담이나 지도가 효과를 발휘하려면 라포(rapport)가 먼저 형성되어야 함.

**핵심 원리:**
- 라포 없이 조언하면 → "쌤은 몰라요"
- 라포 있으면 → "쌤이 그러시면 해볼게요"

**기록할 상황:**
- 학생과 처음 개인적 대화를 나눈 순간
- 학생의 관심사를 알게 된 순간
- "선생님한테만 말하는 건데요..." 같은 신뢰 표현

## 🔗 관련 학생

*백링크에서 자동 수집됨*

| 학생 | 상황 | 계기 |
|:--|:--|:--|
| [[정수아]] | 9/12 쉬는 시간 첫 대화 | 그래프 뷰에서 고립된 점 발견 후 의도적 접근 |
| [[김민준]] | 4/2 개별 상담 | 영어 시간 패턴 발견 후 면담 |
| [[강도현]] | 3/20 개별 면담 | 다툼 이후 가정환경 파악 |

### [[정수아]] 라포 형성 사례 (10장)

**배경:**
- 그래프 뷰에서 1년간 연결 0개인 점 발견
- "조용하고 착해서 완전히 놓쳤구나" 자각

**첫 접촉:**
```
9/12 (월) 쉬는 시간
- "수아야, 요즘 뭐 재미있는 거 있어?"
- 처음엔 당황 → 웹툰 〈재혼황후〉 이야기
- 의외로 말이 많음
- "그동안 내가 기회를 안 준 건가?"
```

**후속 효과:**
- 관심을 가지니까 기록이 생김
- 기록이 생기니까 연결이 생김
- 9월부터 리더십 발휘 장면 다수 발견

## 💡 라포 형성 전략

### 1단계: 관심사 파악

| 접근 방법 | 예시 질문 |
|:--|:--|
| 취미 | "요즘 뭐 재미있는 거 있어?" |
| 미디어 | "요즘 무슨 유튜브 봐?" |
| 게임 | "요즘 무슨 게임 해?" |
| 연예인 | "좋아하는 가수 있어?" |

**Tip:** 정답을 맞추려 하지 말고, **관심을 보이는 것** 자체가 중요

### 2단계: 기억하고 언급하기

```
첫 대화: "웹툰 좋아하는구나"
다음 주: "수아야, 재혼황후 새 화 나왔어?"
```

→ "선생님이 내 얘기를 기억하시네!" → 신뢰 상승

### 3단계: 비공식적 접촉 늘리기

| 장소 | 방법 |
|:--|:--|
| 복도 | 지나가며 눈 마주치고 웃기 |
| 급식실 | "맛있어?" 한마디 |
| 쉬는 시간 | 자리 근처 지나가며 말 걸기 |

### 4단계: 경청하기

- 말 끊지 않기
- 판단하지 않기
- "그랬구나" 반응

## ⚠️ 주의사항

| ❌ 피할 것 | ✅ 권장하는 것 |
|:--|:--|
| 갑자기 과한 관심 | 자연스럽게 조금씩 |
| 다른 학생 앞에서 특별 대우 | 1:1 상황 활용 |
| "왜 맨날 조용해?" | "요즘 어때?" |
| 대답 강요 | 침묵도 허용 |
