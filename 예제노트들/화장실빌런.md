# 화장실빌런

## 📌 정의

수업 시간만 되면 화장실을 간다며 자리를 이탈하는 패턴. 단순한 생리 현상이 아니라 **특정 수업/상황 회피**일 가능성이 높음.

**판별 기준:**
- 특정 요일/교시에만 반복되는가?
- 특정 과목 시간에만 발생하는가?
- 빈도가 주 1회 이상인가?

---

## 🔗 관련 학생

*백링크에서 자동 수집됨*

| 학생 | 패턴 | 원인 |
|:--|:--|:--|
| [[김민준]] | 매주 목요일 2교시 영어 시간 | 영어 듣기평가 불안 |
| [[이준우]] | 금요일 5교시 수학 시간 | (확인 필요) |
| [[박서준]] | 월요일 1교시 아침자습 시간 | (확인 필요) |

### [[김민준]] 상세 사례 (5장)

**발견 과정:**
```
3/07: 2교시 영어, 화장실 (15분 소요)
3/14: 2교시 영어, 수업 시작 5분 만에 화장실
3/21: 2교시 영어, 배 아프다며 보건실
3/28: 2교시 영어, 또 화장실
```

**패턴 발견:** 4주간 기록을 모으니 공통점이 보임
- 요일: **목요일**
- 교시: **2교시**
- 과목: **영어**

**원인 파악:** 상담 결과, 영어 듣기평가만 나오면 머리가 하얘진다고 함

**해결:** 듣기 스크립트 미리 제공 → 4월부터 화장실 안 감

---

## 💡 대응 전략

### 1단계: 패턴 기록
- 화장실 갈 때마다 **날짜, 교시, 과목** 기록
- 최소 3~4회 데이터가 모여야 패턴이 보임

### 2단계: 패턴 분석
| 질문 | 확인할 것 |
|:--|:--|
| 특정 요일인가? | 월요일? 금요일? |
| 특정 교시인가? | 1교시? 4교시? |
| 특정 과목인가? | 영어? 수학? |
| 특정 활동인가? | 발표? 시험? 모둠활동? |

### 3단계: 원인 탐색
- "왜 자꾸 화장실 가니?" ❌ (훈계)
- "혹시 영어 시간이 힘드니?" ✅ (본질적 질문)

### 4단계: 개별 개입
- 과목 불안 → 난이도 조절, 사전 자료 제공
- 발표 불안 → 소그룹부터 시작
- 대인 불안 → 자리 배치 조정
