# 학급경영

## 📌 정의

담임교사가 학급을 운영하며 수행하는 모든 활동. 학생 관리, 생활지도, 환경 조성, 학부모 소통, 행정 업무 등을 포함.

> "수업만 잘하면 되는 게 아니라, 30명이 1년간 함께 살아가는 공간을 만드는 일"

---

## 💡 학급경영 핵심 원칙

### 1. 기록한다
- 문제가 생기면 **당일 기록**
- `[[학생이름]]` 링크로 연결
- 나중에 패턴 파악, 민원 대응 근거

### 2. 일관성 있게
- 규칙은 예외 없이 적용
- "오늘은 봐줄게"가 쌓이면 규칙이 무너짐
- 학생들은 일관성에서 안정감을 느낌

### 3. 조용한 아이도 본다
- 문제 일으키는 아이는 자동으로 보임
- **조용한 아이**가 더 위험 ([[정수아]] 사례)
- 그래프 뷰로 "고립된 점" 확인

### 4. 관계가 먼저
- [[라포형성]] 없이 지도 없음
- "쌤이 나한테 관심 있구나" 느끼게
- 규칙 위반해도 관계는 유지

---

## 📊 학급경영 연간 흐름

```
3월: 규칙 수립 → 관계 형성
     ↓
4~6월: 갈등 발생 → 개입 → 조정
     ↓
7월: 1학기 정리
     ↓
9~11월: 심화 → 성장
     ↓
12월: 생기부 → 마무리
```

## 🔗 학급경영과 기록의 관계

| 기록 없이 | 기록 있으면 |
|:--|:--|
| "걔 원래 그래" | 패턴 분석 → 원인 파악 |
| 민원 오면 당황 | 근거 제시 |
| 12월 생기부 막막 | 검색해서 조립 |
| 조용한 아이 놓침 | 그래프 뷰로 발견 |
