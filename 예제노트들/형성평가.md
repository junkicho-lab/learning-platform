# 형성평가

## 📌 정의

수업 중간이나 마무리 단계에서 학생들의 이해도를 확인하는 짧은 평가. 점수 매기기보다 **"어디까지 알고, 뭘 모르는지"** 파악이 목적.

---

## 📅 적용 기록

### 2024년
- **방식:** 수업 끝나고 "질문 있어요?" (대부분 침묵)
- **결과:** ⭐⭐ (2/5)
- **문제점:**
  - 이해 못 한 학생도 질문 안 함
  - 다음 시간에 가서야 "몰랐어요" 발견

### 2025년
- **방식:** OX 퀴즈 + 손들기
- **결과:** ⭐⭐⭐⭐ (4/5)
- **개선점:**
  - "O라고 생각하면 손!" → 전원 참여
  - 오답 많으면 바로 재설명
  - 누가 모르는지 눈으로 확인 가능

### 2026년
- **방식:** 디지털 도구 + 익명 응답
- **결과:** ⭐⭐⭐⭐⭐ (5/5)
- **성공 요인:**
  - 카훗/멘티미터로 익명 응답
  - 틀려도 창피하지 않음 → 솔직한 응답
  - 실시간 통계로 "70%가 이거 모르네" 파악

---

## 💡 핵심 노하우

1. **짧게:** 5문제 이내, 5분 이내
2. **전원 참여:** 손들기, 화이트보드, 디지털 도구
3. **즉각 피드백:** 틀린 문제 바로 설명
4. **부담 없이:** 점수 반영 ❌, 이해도 확인 ✅
5. **기록:** 오답률 높은 문제 → 다음 시간 복습

---

## ⚠️ 주의사항

- 어려운 문제만 내면 자신감 하락
- 쉬운 문제 → 중간 → 어려운 순서로
- "맞힌 사람?" 하면 못 맞힌 학생 위축됨 → 익명이 좋음

---

## 📊 형성평가 유형

| 유형 | 도구 | 장점 |
|:--|:--|:--|
| OX 퀴즈 | 손들기 | 준비 필요 없음 |
| 미니 화이트보드 | 개인 보드 | 전원 답 한눈에 |
| 카훗/멘티미터 | 스마트폰 | 익명, 실시간 통계 |
| 출구 티켓 | 종이 | 수업 끝에 한 줄 쓰고 제출 |
| 엄지척 | 손가락 | 3초 만에 분위기 파악 |

