# 갈등조정

## 📌 정의

학생들 사이의 갈등 상황을 중재하는 역량. 교사의 개입뿐 아니라 **학생이 또래 갈등을 조정하는 모습**도 포함.

**기록할 상황:**
- 학생이 친구들 사이 갈등을 중재한 장면
- 모둠활동에서 의견 충돌을 조율한 경우
- 학급회의에서 합의를 이끌어낸 경우

**생기부 활용:** 리더십, 의사소통 능력의 좋은 근거가 됨

### [[정수아]] 갈등 중재 사례 (12장)

```
9/25 (목) 학급회의
- 안건: 수학여행 조 편성
- 상황: 친한 친구끼리만 같은 조 하려고 갈등 발생
- 정수아 역할: 
  - "랜덤으로 정하고, 대신 자유시간 때 원하는 친구랑 다니면 어때요?"
  - 양쪽 의견 절충안 제시
- 결과: 합의 도출
```

**교사 관찰 메모:**
> 조용하지만 리더십 있음. 목소리를 높이지 않고도 정리함.

## 💡 갈등 중재를 잘하는 학생 특징

| 특징 | 설명 |
|:--|:--|
| 경청 | 양쪽 이야기를 끝까지 들음 |
| 중립 | 한쪽 편을 들지 않음 |
| 절충안 제시 | "이렇게 하면 어때?" |
| 감정 조절 | 본인이 흥분하지 않음 |

## 💡 교사의 갈등 중재 전략

### 1단계: 분리 및 안정화
- 흥분 상태에서는 중재 불가
- 물리적 거리 확보 후 진정 시간

### 2단계: 개별 청취
- 각자 입장을 **따로** 듣기
- "그래서 네 기분이 어땠어?"

### 3단계: 공통점 찾기
- "둘 다 ~~한 건 같네"
- 공통 목표 확인

### 4단계: 해결책 함께 찾기
- 교사가 정답 제시 ❌
- "어떻게 하면 좋을까?" ✅

### 5단계: 후속 관찰
- 일주일 후 관계 확인
- [[교우관계]] 태그로 기록
