# 가정통신문 상용구 모음

## 📌 인사말 (계절별)

### 봄
따스한 봄바람이 불어오는 요즘, 학부모님 가정에 평안을 기원합니다.

새 학기를 맞아 학부모님의 가정에 건강과 행복이 함께하시길 바랍니다.

봄꽃이 만개한 계절, 댁내 두루 평안하시길 바랍니다.

### 여름
무더운 날씨에 건강 유의하시기 바라며, 가정에 행복이 가득하시길 바랍니다.

장마철 건강에 유의하시고, 가정에 평안이 깃들기를 바랍니다.

여름방학을 앞두고 학부모님의 가정에 기쁨이 가득하시길 기원합니다.

### 가을
선선한 바람이 부는 가을, 댁내 평안하시길 바랍니다.

결실의 계절을 맞아 가정에 풍요로움이 가득하시길 바랍니다.

단풍이 물드는 계절, 학부모님의 가정에 행복이 함께하시길 바랍니다.

### 겨울
추운 날씨에 건강하시길 바라며, 새해에도 좋은 일만 가득하시길 기원합니다.

한 해를 마무리하는 시기, 가정에 평안과 건강이 함께하시길 바랍니다.

겨울방학을 앞두고 학부모님의 가정에 따뜻함이 가득하시길 바랍니다.

## 📌 마무리 문구

### 협조 요청형
가정에서도 많은 관심과 협조 부탁드립니다.

학부모님의 적극적인 협조를 부탁드립니다.

가정에서의 지도 협조 부탁드립니다.

### 문의 안내형
기타 문의사항은 담임교사(☎ 031-000-0000)에게 연락 바랍니다.

궁금하신 점은 언제든 담임교사에게 문의해 주시기 바랍니다.

자세한 사항은 학교 홈페이지 또는 담임교사에게 문의 바랍니다.

### 감사형
항상 학교 교육에 관심 가져주셔서 감사합니다.

학부모님의 변함없는 성원에 감사드립니다.

늘 학교를 믿고 응원해 주셔서 감사합니다.

## 📌 자주 쓰는 안내 문구

### 준비물 안내
```
※ 준비물
  - 
  - 
  
※ 복장: 

※ 집합 시간 및 장소:
```

### 비용 안내
```
※ 참가비: ____________원

※ 납부 기한: 2026년    월    일까지

※ 납부 방법
  [ ] 학교 계좌 입금 (○○은행 000-000-000000, 예금주: ○○중학교)
  [ ] 담임교사에게 전달

※ 입금 시 학생 이름으로 입금 바랍니다.
```

### 일정 안내
```
※ 일시: 2026년    월    일 (   요일)

※ 장소: 

※ 일정
  - 00:00 집합
  - 00:00 출발
  - 00:00 도착
  - 00:00 귀가
```
