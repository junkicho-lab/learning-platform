# 행정실

## 📌 주요 업무

- **시설 관리:** 교실 수리, 냉난방, 전기/수도
- **물품 대여:** 텐트, 테이블, 의자, 확성기
- **예산 집행:** 학급비, 행사비 지출
- **문서 발송:** 공문, 가정통신문 인쇄
- **차량 지원:** 버스 대절 계약, 출장 차량

- **시설 담당:** 내선 103

## ✅ 요청 시 체크리스트

### 물품 대여 (텐트, 테이블 등)
- [ ] **D-14 이전** 요청 (급한 요청 어려움)
- [ ] 대여 물품명, 수량 명시
- [ ] 사용 일시, 장소 명시
- [ ] 반납 일정 확인

### 예산 집행
- [ ] 품의서 작성
- [ ] 결재 완료 후 요청
- [ ] 영수증/세금계산서 제출

### 시설 수리
- [ ] 수리 요청서 작성 (교무실 양식)
- [ ] 긴급 시 직접 방문

## 💡 꿀팁

- **급한 요청은 어렵다** → 최소 2주 전 요청
- **구두 요청 ❌** → 메일 또는 요청서로 기록 남기기
- **예산 문의** → 학년 초에 배정액 확인해두기
- **텐트 인기 많음** → 체육대회 시즌엔 일찍 선점
