# 수업방해

## 📌 정의

수업 진행을 방해하는 행동 전반. 떠들기, 돌아다니기, 딴짓, 친구 건드리기 등.

## 🔗 관련 학생

*백링크에서 자동 수집됨*

- (해당 학생 발생 시 기록)

## 💡 대응 전략

- **즉각 대응:** 이름 부르기 → 눈 마주치기 → 가까이 가기
- **패턴 파악:** 특정 시간대/과목에 집중되는지 확인
- **원인 탐색:** 수업 난이도? 집중력 문제? 관심 끌기?
- **좌석 배치:** 산만한 학생은 앞자리 또는 분리
