# 학부모상담 매뉴얼

## 📌 개요

학기별 학부모 상담 주간 운영 및 수시 상담 대응을 위한 매뉴얼.

> **"30명 학부모와 각각 15분. 기록 없이는 '요즘 열심히 해요~'밖에 할 말이 없다."**

---

## 📅 상담 주간 일정

- **1학기:** 4월 말 ~ 5월 초
- **2학기:** 10월 말 ~ 11월 초
- **상담 시간:** 1인당 15~20분
- **방식:** 대면 / 전화 / 화상 (학부모 선택)

---

## ✅ 상담 전 체크리스트

### D-7 (일주일 전)
- [ ] 전체 학생 노트 훑어보기
- [ ] `#상담필요` 태그 검색 → 관심 학생 파악
- [ ] 상담 일정표 확정 및 안내
- [ ] 상담 장소 확보 (교실 또는 상담실)

### D-1 (전날)
- [ ] 내일 상담할 학생 5명 노트 확인
- [ ] 각 학생별 **칭찬할 점 1개** 메모
- [ ] 각 학생별 **성장 포인트 1개** 메모
- [ ] 민감한 사안 있는 학생 별도 준비 ([[가정환경]] 등)

### 당일
- [ ] 학생 노트를 **모니터에 띄워두고** 상담
- [ ] 상담 후 **당일 기록** (데일리 노트)
- [ ] `#상담완료` 태그 추가
- [ ] 후속 조치 필요 시 `#후속필요` 태그

## 💡 상담 팁

### 1. 칭찬으로 시작하기

> **"어머니, 민준이가 요즘 영어 시간에 집중을 잘해요."**

- 첫 마디가 긍정적이면 분위기가 달라짐
- 칭찬할 거리는 **데일리 노트에서 검색**
- `[[김민준]] 칭찬` 또는 `#행특 [[김민준]]`

### 2. 구체적 사례 제시하기

| ❌ 애매한 표현 | ✅ 구체적 표현 |
|:--|:--|
| "열심히 해요" | "5월에 발표를 3번 했어요" |
| "친구들과 잘 지내요" | "모둠활동에서 조장 역할을 맡았어요" |
| "조금 산만해요" | "3월에 비해 집중 시간이 늘었어요" |

### 3. 가정 협조 요청하기

> **"가정에서도 이 부분을 함께 봐주시면 좋겠어요."**

- 일방적 요구 ❌ → 협력 요청 ✅
- "혼내주세요" ❌ → "관심 가져주세요" ✅
- 구체적 행동 제안: "하루 10분 독서 함께 해주세요"

### 4. 기록하며 상담하기

- 상담 중 핵심 키워드 메모
- 상담 직후 10분 내 정리 (기억이 생생할 때)
- 합의 사항은 **문자로 재확인** (증거 남기기)

## 📊 학생별 상담 포인트

### [[김민준]]
- **칭찬:** 2학기 영어 시간 집중력 향상
- **공유:** [[영어회피]] 패턴 → 듣기 스크립트 지원 효과
- **협조:** 가정에서 영어 듣기 연습 격려

### [[정수아]]
- **칭찬:** 학급회의 갈등 중재, 급식 도우미 솔선수범
- **공유:** 조용하지만 영향력 있는 리더십
- **주의:** "조용해서 잘 모르겠어요"라고 하지 말 것!

### [[강도현]]
- **칭찬:** 체육 활동 리더십, 후배 배려
- **공유:** [[교우관계]] 개선 과정 (3월 vs 현재)
- **협조:** 가정에서 정서적 지지

### [[최예진]]
- **칭찬:** 수업 태도 성실
- **민감:** [[가정환경]] 관련 (조심스럽게)
- **협조:** 학교에서 관심 갖고 있다는 안심 전달

## ⚠️ 민감한 상담 시 주의사항

### [[가정환경]] 관련
- 직접 묻기보다 "요즘 집에서는 어떠세요?" 열린 질문
- 학부모가 먼저 이야기하면 경청
- 기록은 `_비공개` 폴더에 별도 저장

### 성적 하락 관련
- "왜 떨어졌는지" 추궁 ❌
- "어떻게 도와드릴까요" 협력 자세
- 구체적 지원 방안 제시

### 교우관계 갈등
- 한쪽 편만 들지 않기
- "사실 확인 후 연락드리겠습니다"
- 상대방 학부모에게도 동일한 태도

## 📞 수시 상담 (전화) 대응

### 전화가 오면
1. **옵시디언 켜기** (학생 노트 준비)
2. **`Ctrl+Shift+F`** → 학생 이름 검색
3. 기록 확인하며 대화

### 민원성 전화 시
- [[학부모 민원 대응법]] 참고
- 3단계 팩트 전환 대화법 적용
