# 비계설정

## 📌 정의

**건축 현장의 비계(발판)처럼, 학습자가 혼자 할 수 없는 과제를 해낼 수 있도록 임시 지원을 제공하는 것.**

비고츠키(Vygotsky)의 **근접발달영역(ZPD)** 개념에서 유래. 학습자가 "혼자서는 못 하지만, 도움 받으면 할 수 있는" 영역에서 적절한 도움(비계)을 제공하고, 실력이 늘면 점차 비계를 제거하는 교수 전략.

---

## 🏫 교실에서의 의미

> **"너무 어려워요" → "이 정도면 할 수 있을 것 같아요"**

이 전환을 만드는 게 비계설정입니다.

```
[혼자 할 수 있는 영역]
        ↓
[비계 있으면 할 수 있는 영역] ← 여기서 가르친다!
        ↓
[아무리 도와줘도 못 하는 영역]
```

**핵심 원리:**
1. 학생 수준 파악
2. **살짝 위** 과제 제시
3. 적절한 도움(비계) 제공
4. 성공하면 비계 **점차 제거**
5. 결국 혼자서도 할 수 있게

## 🔗 관련 사례

### [[김민준]] - 듣기 스크립트라는 비계

**문제 상황 (3월)**
- 영어 듣기평가만 나오면 "저 어차피 못해요"
- 듣기 음원 → 머리 하얘짐 → 포기 → [[화장실빌런]]
- **비계 없이 바로 도전** → 실패 → [[자기효능감]] 하락

**비계 설정 (4월)**
| 비계 유형 | 적용 내용 |
|:--|:--|
| **사전 자료** | 듣기 스크립트 미리 제공 |
| **난이도 조절** | 쉬운 1번 문제만 먼저 풀기 |
| **심리적 지지** | "틀려도 괜찮아" 분위기 |

**비계 점진적 제거 (5~9월)**
1. 스크립트 전체 제공 → 핵심 단어만 제공
2. 핵심 단어 제공 → 첫 문장만 힌트
3. 힌트 없이 도전 → "쌤, 저 70점 나왔어요!"

**결과**
- 화장실 이탈 횟수 급감
- 스스로 어려운 문제에 도전 시작
- [[자기효능감]] 상승 → 선순환

## 💡 실천 전략

### 1. 학생 수준 먼저 파악
- 어디까지 혼자 할 수 있는지 확인
- "이건 할 수 있어?" 간단한 진단
- 백링크 기록으로 패턴 파악

### 2. 비계 유형 선택

| 비계 유형 | 예시 |
|:--|:--|
| **사전 자료** | 스크립트, 핵심 단어, 예시 답안 |
| **단계 분해** | 큰 과제를 작은 단계로 쪼개기 |
| **힌트 제공** | 첫 글자, 공식, 풀이 방향 |
| **모델링** | 교사가 먼저 시범 보이기 |
| **또래 도움** | 잘하는 친구와 짝짓기 |
| **도구 제공** | 계산기, 사전, 참고 자료 |

### 3. 적절한 수준으로 조절
- 너무 많은 비계 ❌ → 의존성 생김
- 너무 적은 비계 ❌ → 좌절감
- **"조금만 노력하면 할 수 있는"** 수준

### 4. 점진적 제거 (Fading)
- 성공하면 비계 한 단계씩 줄이기
- "이번엔 힌트 없이 해볼까?"
- 최종 목표: 혼자서도 할 수 있게

### 5. 성공 경험 강조
- 비계 있어도 **"네가 해낸 거야"**
- 도움받았다고 가치 떨어지는 거 아님
- [[자기효능감]] 연결

---

## 📊 비계설정 단계 예시 (듣기평가)

| 단계 | 비계 수준 | 민준이 상태 |
|:--|:--|:--|
| 1단계 | 스크립트 전체 + 해석 | "이건 읽으면 되네" |
| 2단계 | 스크립트만 (해석 없음) | "단어는 알겠어" |
| 3단계 | 핵심 단어만 | "이 단어 나오면 집중!" |
| 4단계 | 첫 문장 힌트만 | "시작은 알겠다" |
| 5단계 | 비계 없음 | "70점이에요!" |

---

## ⚠️ 주의사항

- **과잉 비계:** 학생이 의존하게 됨 → 비계 제거 시점 놓치지 말 것
- **과소 비계:** 실패 반복 → 자기효능감 하락
- **일률적 비계:** 학생마다 필요한 비계가 다름 → 개별화 필요
