# 보건실투어

## 📌 정의

"배 아파요", "머리 아파요"를 반복하며 보건실로 향하는 패턴. 진짜 아픈 경우도 있지만, **수업 회피 목적**일 가능성을 항상 염두에 둬야 함.

**판별 포인트:**
| 구분 | 진짜 아픔 | 회피 목적 |
|:--|:--|:--|
| 빈도 | 불규칙 | 특정 요일/교시에 집중 |
| 증상 | 구체적 | 막연함 ("그냥 아파요") |
| 표정 | 힘들어 보임 | 눈치 봄 |
| 복귀 | 쉬고 나면 나아짐 | 수업 끝나면 멀쩡 |

## 🔗 관련 학생

*백링크에서 자동 수집됨*

| 학생 | 패턴 | 증상 | 의심 원인 |
|:--|:--|:--|:--|
| [[김민준]] | 3/21 목요일 2교시 영어 | "배 아파요" | 영어 듣기평가 불안 (확인됨) |

### [[김민준]] 사례

```
3/21 (목) 2교시 영어
- "배 아파요. 보건실 가도 돼요?"
- 보건실에서 20분 후 복귀
- 복귀 후 멀쩡함
- 참고: 직전 3주간 같은 시간에 화장실 반복
```

**결론:** 배가 아픈 게 아니라 영어 시간이 힘든 것

## 💡 대응 전략

### 1단계: 일단 보내주기

첫 1~2회는 의심하지 말고 보내주세요. 진짜 아픈 경우도 있으니까요.

### 2단계: 기록하기

보건실 갈 때마다 기록:
```markdown
- [[학생이름]] 보건실 (증상: / 교시: / 소요시간: )
```

### 3단계: 패턴 확인

3회 이상 반복되면 아래 체크:
- [ ] 같은 요일인가?
- [ ] 같은 교시인가?
- [ ] 같은 과목인가?
- [ ] 증상이 항상 막연한가?

### 4단계: 보건교사와 협력

> "선생님, 민준이가 목요일 2교시마다 오는데, 패턴이 있어서요. 혹시 진짜 아파 보이나요?"

보건교사의 관찰이 중요한 단서가 됨.

### 5단계: 원인 해결

| 원인 | 대응 |
|:--|:--|
| 특정 과목 불안 | 난이도 조절, 사전 자료 제공 |
| 특정 활동 회피 | 역할 조정, 단계적 노출 |
| 대인관계 문제 | 자리 배치, 모둠 조정 |
| 실제 건강 문제 | 가정 연계, 병원 권유 |

## ⚠️ 주의사항

**절대 하지 말 것:**
- "꾀병 아니야?" ❌
- "또 보건실이야?" (한숨) ❌
- 다른 학생들 앞에서 의심하기 ❌

**이유:** 진짜 아픈 날 못 가게 되면 큰 문제. 일단 보내고, 패턴은 기록으로 확인.
