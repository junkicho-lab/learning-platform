# 완전 초보자용 효능감 획득 실습 노트

---

## 📌 시작하기 전에 읽어주세요

### 이 실습의 규칙

```
✅ 하루에 딱 1개 장만 합니다
✅ 각 장에서 딱 1개 미션만 성공하면 됩니다
✅ 완벽하게 안 해도 됩니다
✅ 틀려도 됩니다
✅ 3분 넘게 걸리면 그냥 넘어가세요
```

### 어려운 말 미리 번역

| 어려운 말 | 쉬운 말 |
|----------|--------|
| 데일리 노트 | 오늘 일기 |
| 링크 `[[ ]]` | 파란 글씨로 바꾸기 |
| 백링크 | 예전에 쓴 것들이 모여있는 곳 |
| 태그 `#` | 스티커 붙이기 |
| MOC | 목차 |

# 🌱 1장 실습: 첫 번째 기록 남기기

## 오늘의 미션
> **"오늘 학교에서 있었던 일 딱 1줄 적기"**

### Step 1: 새 노트 만들기
1. 옵시디언을 엽니다
2. 왼쪽에서 📄 아이콘(새 노트)을 누릅니다
3. 제목에 오늘 날짜를 적습니다 (예: `2026-03-14`)

### Step 2: 이것만 복사하세요
```
# 오늘 하루

## 기억나는 일
- 

## 내일 할 일
- [ ] 
```

### Step 3: 딱 1줄만 채우세요
```
# 오늘 하루

## 기억나는 일
- 민준이가 수업 시간에 발표를 잘했다

## 내일 할 일
- [ ] 
```

### ✅ 성공 체크
```
□ 오늘 날짜로 노트를 만들었다
□ 1줄이라도 적었다
```

**2개 다 체크했으면 → 1장 성공! 🎉**

---

# 🌱 2장 실습: 이름에 파란색 입히기

## 오늘의 미션
> **"학생 이름을 파란 글씨로 바꾸기"**

### Step 1~2: 어제 노트 열고 학생 이름 찾기

### Step 3: 대괄호 씌우기
**Before:**
```
- 민준이가 수업 시간에 발표를 잘했다
```

**After:**
```
- [[민준]]이가 수업 시간에 발표를 잘했다
```

### ✅ 성공 체크
```
□ 학생 이름에 [[ ]]를 씌웠다
□ 파란색으로 변했다
```

**2개 다 체크했으면 → 2장 성공! 🎉**

---

# 🌱 3장 실습: 할 일 체크박스 만들기

## 오늘의 미션
> **"할 일 1개 적고, 완료하면 체크하기"**

### 체크박스 만들기
```
- [ ] 민준이 어머니께 전화하기
```

### 완료하면
```
- [x] 민준이 어머니께 전화하기
```

# 🌱 4장 실습: 파란 글씨 눌러보기

## 오늘의 미션
> **"파란 글씨를 눌러서 새 노트로 이동하기"**

- `[[민준]]` 클릭 → 민준 노트로 이동
- 돌아가기: 왼쪽 위 **← 화살표**

# 🌱 5장 실습: 같은 학생 여러 번 기록하기

## 오늘의 미션
> **"같은 학생을 3일간 기록하고, 기록이 모이는 걸 확인하기"**

### 3일치 노트
**노트 1: `2026-03-07`**
```
- [[민준]]이가 2교시에 화장실 갔다
```

**노트 2: `2026-03-14`**
```
- [[민준]]이가 또 2교시에 화장실 갔다
```

**노트 3: `2026-03-21`**
```
- [[민준]]이가 또또 2교시에 화장실 갔다. 3주 연속이다.
```

### 패턴 발견!
→ **"민준이는 매주 2교시에 화장실을 간다!"**

# 🆘 자주 하는 실수와 해결법

| 문제 | 해결법 |
|:---|:---|
| 파란색이 안 돼요! | `[민준]` → `[[민준]]` (대괄호 2개씩!) |
| 저장 버튼이 없어요! | 자동 저장됨. 그냥 닫아도 OK |
| 화면이 바뀌었어요! | ← 화살표로 돌아가기 |
| 그래프 뷰가 무서워요! | 정상! 점이 많을수록 좋은 것 |
