# Part 2. 마크다운 기초 15분

> "단순함이 궁극의 정교함이다" - 레오나르도 다빈치

마크다운은 **젓가락**입니다. 처음엔 포크와 나이프가 편해 보이지만, 익숙해지면 젓가락 하나로 모든 요리를 먹을 수 있습니다. #, ## 은 측정자, # 은 견출지, \[\[ \]\]는 실과 바늘. 3가지 도구로 문서라는 옷을 짜는 것입니다.

---

## 2.1 마크다운이 뭔가요?

### 🤔 한글/워드 없이 글 꾸미기?


**일반적인 방법 (한글/워드)**:
```
1. 텍스트 드래그
2. 굵게 버튼 클릭
3. 크기 선택
4. 색상 변경
→ 번거로움!
```

**마크다운 방법**:
```
**굵게**만 입력
→ 자동으로 굵게 표시!
→ 초간단!
```

### 💡 핵심 개념

```
마크다운 = 기호로 글 꾸미기

비유:
이모티콘 ^^ → 😊
마크다운 **굵게** → 굵게
```

**장점**:
- ✅ 마우스 안 써도 됨
- ✅ 빠름 (버튼 찾을 필요 없음)
- ✅ 어떤 프로그램에서든 동일
- ✅ 10년 후에도 열림

## 4.3 실전 예시: 업무일지
### A. 제목 만들기

**문법**:
```markdown
# 큰 제목 (제일 큼)
## 중간 제목
### 작은 제목
```

**실제 표시**:
# 큰 제목 (제일 큼)
## 중간 제목
### 작은 제목

**💡 팁**:
`#` 기호는 글자를 **제목**으로 만들어줍니다. `#`의 개수에 따라 글자 크기가 달라져요.
- `#` 뒤에 **띄어쓰기 필수!**
- `#`: 가장 큰 제목 (h1)
- `##`: 중간 제목 (h2)
- `###`: 작은 제목 (h3)
- `####`: 더 작은 제목 (h4) ... 보통 `#`은 6개까지 사용합니다.

이 기호를 쓰면 문서의 구조를 한눈에 파악하기 좋아집니다.

**실습 예시**:
```markdown
# 2024-10-12 업무일지
## 오전 업무
### 1교시 수업
```

### B. 링크 걸기 (가장 중요!)

**문법**:
```markdown
[[링크할 노트 이름]]
```

**실제 표시**:
[[김민준]] ← 클릭하면 김민준 노트로 이동

**💡 팁**:
- 대괄호 2개 사이에 이름
- 파일이 없어도 됨 (자동 생성)
- 이게 옵시디언의 핵심!

이 문법은 옵시디언 같은 앱에서 아주 강력한 기능입니다. 단어를 `[[ ]]`(대괄호 두 개)로 감싸면 해당 단어와 똑같은 이름의 **다른 노트로 바로 이동하는 링크**가 만들어집니다. 예를 들어, `[[김민준]]`을 클릭하면 '김민준' 학생에 대한 모든 기록을 모아둔 노트로 바로 넘어갈 수 있죠. 흩어진 정보를 연결하는 핵심적인 역할을 합니다.

**실습 예시**:
```markdown
오늘 [[김민준]] 학생과 상담
[[수학 수업]] 자료 정리
[[생활기록부]] 작성 시작
```

### C. 태그 달기

**문법**:
```markdown
- [ ] #학생지도 #2학년 #김민준
```

**실제 표시**:
- [ ] #학생지도 #2학년 #김민준

**💡 팁**:
태그는 노트를 분류하는 라벨입니다.
태그를 누르면 그 태그가 달린 노트만 모아볼 수 있습니다. #김민준 을 누르면 12개 노트가 나옵니다. #학습지도 를 누르면 34개가 나옵니다. 이것이 태그의 힘입니다.

### D. 목록 만들기

**문법**:
```markdown
- 첫 번째 항목
- 두 번째 항목
- 세 번째 항목
```

**실제 표시**:
- 첫 번째 항목
- 두 번째 항목
- 세 번째 항목

**💡 팁**:
- `-`(하이픈)이나 `*`(별표)를 문장 앞에 붙이면 **글머리 기호 목록**이 만들어집니다. 순서가 없는 목록을 정리할 때 아주 유용해요.
- 번호 목록은 `1.` 사용

**최초의노트시**:
```markdown
## 오늘의 할 일
- 수업 준비
- 생기부 작성
- 학부모 상담
```

### F. 체크박스

**문법**:
```markdown
- [ ] 할 일 (안 함)
- [x] 완료된 일 (완료)
```

**실제 표시**:
- [ ] 할 일 (안 함)
- [x] 완료된 일 (완료)

**💡 팁**:
목록 기호(`-`) 뒤에 `[ ]`를 추가하면 **체크박스**가 있는 할 일 목록을 만들 수 있습니다.
- 클릭하면 체크/해제 전환
- `[ ]`: 아직 완료되지 않은 일
- `[x]`: 완료된 일
일을 체계적으로 관리할 때 정말 편리한 기능입니다.

**실습 예시**:
```markdown
## 내일 할 일

- [ ] [[중2-5반]] 수업
- [ ] [[체험학습]] 최종 점검
- [ ] [[생기부]] 3명 더
```

### G. 굵게, 기울임

**문법**:
```markdown
**굵게**
*기울임*
***굵게 + 기울임***
```

**실제 표시**:
- **굵게**
- *기울임*
- ***굵게 + 기울임***

**💡 팁**:
- `**단어**` 또는 `__단어__`: 별표나 언더바 두 개로 단어를 감싸면 **글자를 굵게(Bold)** 만들어 강조합니다.
- `*단어*` 또는 `_단어_`: 별표나 언더바 한 개로 감싸면 _글자가 기울어집니다(Italic)_.
- `***단어***`: 별표 세 개로 감싸면 **_글자가 굵어지고 기울어집니다_**.

**실습 예시**:
```markdown
**중요**: 내일까지 제출
*참고*: 선택 사항
***필독***: 꼭 읽어주세요
```
